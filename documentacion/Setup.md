# Guía de Instalación y Uso

La aplicación ahora está dividida en Backend y Frontend. Para usarla, debes ejecutar ambos.

## 1. Backend (Python)
Abre una terminal en la carpeta raíz del proyecto y entra a `backend`:
```powershell
cd backend
pip install -r requirements.txt
python app.py
```
*El backend quedará corriendo en http://localhost:8000*

## 2. Frontend (Next.js)
Abre otra terminal en la carpeta raíz del proyecto y entra a `frontend`:
```powershell
cd frontend
npm install
npm run dev
```
*El frontend quedará corriendo en http://localhost:3000*

## Uso Diario
Solo entra a `http://localhost:3000` en tu navegador. Puedes guardar ese link en tus marcadores.
